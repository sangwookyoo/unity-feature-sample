# WindowHelper 클래스

`WindowHelper` 클래스는 Unity에서 윈도우 창을 제어하는 기능을 제공합니다. 이 클래스는 Win32 API를 사용하여 윈도우의 크기, 위치, 스타일 등을 변경할 수 있으며, Unity 게임 실행 시 윈도우를 프레임이 있는 창 또는 프레임이 없는 창으로 설정할 수 있습니다.

## 주요 기능
- **윈도우 스타일 설정**: 윈도우를 프레임이 있는 창 또는 프레임이 없는 창으로 전환합니다.
- **윈도우 크기 및 위치 변경**: 윈도우의 위치를 이동하고 크기를 조정합니다.
- **윈도우 최소화 및 최대화**: 윈도우를 최소화하거나 최대화하며, 복원 기능도 제공합니다.

## 메서드 설명
1. **SetFramelessWindow()**
    - 윈도우를 프레임이 없는 형태로 설정합니다.

2. **SetFramedWindow()**
    - 윈도우를 프레임이 있는 기본 창 형태로 설정합니다.

3. **MinimizeWindow()**
    - 현재 활성화된 윈도우를 최소화합니다.

4. **MaximizeWindow()**
    - 현재 활성화된 윈도우를 최대화합니다.

5. **RestoreWindow()**
    - 최소화 또는 최대화된 윈도우를 원래 상태로 복원합니다.

6. **MoveWindowPos(Vector2 posDelta, int newWidth, int newHeight)**
    - 윈도우의 현재 위치를 기준으로 이동시키고, 새 크기를 설정합니다.
